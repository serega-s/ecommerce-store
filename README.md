# Ecommerce Store

### 1. Install requirements from file

```
pip install -r requirements.txt
```

### 2. Make migrations

```
python manage.py makemigrations
```

### 3. Apply Migrations

```
python manage.py migrate
```

### 4. Create superuser (admin)

```
python manage.py createsuperuser
```

### 5. Run server

```
python manage.py runserver
```
